USER FLOW:
══════════

1. App Launch
   ┌─────────┐
   │  Start  │
   └────┬────┘
        │
        ▼
   ┌──────────────┐
   │ Login Screen │  ← hamburger hidden
   └──────┬───────┘
          │ login()
          ▼
   ┌────────────────┐
   │ Show Hamburger │  ← btn_hamburger.opacity = 1
   └────────┬───────┘
            │
            ▼
   ┌────────────────┐
   │ Vitrine Screen │  ← sm.current = 'vitrine'
   └────────────────┘

2. Menu Interaction
   ┌────────────────┐
   │ Click [☰]      │
   └────────┬───────┘
            │ toggle_menu(True)
            ▼
   ┌────────────────┐
   │  Menu Opens    │  ← menu_aberto = True
   │  (slides in)   │
   └────────┬───────┘
            │
            ├─► Click option → Navigate + close menu
            ├─► Click backdrop → Close menu
            └─► Click Sair → Logout

3. Logout
   ┌────────────────┐
   │  Click Sair    │
   └────────┬───────┘
            │ logout()
            ▼
   ┌────────────────┐
   │ Hide Hamburger │  ← btn_hamburger.opacity = 0
   └────────┬───────┘
            │
            ▼
   ┌────────────────┐
   │  Close Menu    │  ← fechar_menu()
   └────────┬───────┘
            │
            ▼
   ┌────────────────┐
   │ Login Screen   │  ← sm.current = 'login'
   └────────────────┘
